Kivy GUI Framework (Python) Installation and use guide

After the installation, open cmd as administrator:

-To open Kivy Virtual Environment:

kivy_venv\Scripts\activate

-To open Kivy samples code:

python kivy_venv\share\kivy-examples\demo\showcase\main.py
